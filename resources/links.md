## Further Learning

### Recommended Path
1. Start: [Webflow Developer Portal](https://developers.webflow.com)
2. Learn: [TypeScript Handbook](https://www.typescriptlang.org/docs/)
3. Practice: [Create simple extension](https://github.com/Webflow-Examples/designer-api-starter)
4. Reference: [Designer API Docs](https://developers.webflow.com/designer/reference/introduction)
5. Build: Create your own app
6. Deploy: Follow [deployment guide](./deployment-checklist.md)
7. Share: Submit to [Webflow marketplace](https://webflow.com/apps)
